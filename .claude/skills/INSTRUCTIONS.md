# Coding Subagent Instructions

This file provides instructions for Claude Code subagents operating within the Product Engine repository.

## Primary Reference

For full system instructions, see the root [`CLAUDE.md`](../../CLAUDE.md).

## Subagent-Specific Notes

- When spawned as a coding subagent, follow all instructions in `CLAUDE.md`
- Skills are located in `_skills/pe-*/SKILL.md`
- Reference files for each skill are in `_skills/pe-*/references/`
- Repository configuration is in `_config/repo.md`
- Always preserve YAML frontmatter in any files you edit
- Do not modify `_prompts/` files unless explicitly instructed
