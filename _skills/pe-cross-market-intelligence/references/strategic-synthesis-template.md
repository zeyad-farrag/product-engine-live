# Strategic Synthesis Template — Cross-Market Intelligence

Use this template during Phase 3 of every cross-market intelligence run.
Complete all three sections in order. The synthesis should be grounded
entirely in evidence from the five analysis frameworks — no new claims
should appear here that aren't already supported by Analysis 1–5.

---

## Top 5 Cross-Market Insights

Rank the 5 most strategically important findings from this cross-market
analysis. Ranking criteria:
1. **Evidence breadth** — findings supported by more markets rank higher
2. **Decision impact** — findings that change what the business should do
3. **Urgency** — findings that require near-term action rank higher than
   long-term observations

| # | Insight | Evidence (markets / data supporting it) | Confidence | Strategic Implication | Recommended Action |
|---|---|---|---|---|---|
| 1 | [insight — specific, evidenced finding] | [which markets / artifact types support it] | [HIGH / MED / LOW] | [what this means for strategy] | [concrete action — initiative, product change, capability to run] |
| 2 | | | | | |
| 3 | | | | | |
| 4 | | | | | |
| 5 | | | | | |

**Confidence levels:**
- `HIGH` — supported by 3+ independent markets / artifact types
- `MEDIUM` — supported by 2 markets or a single high-quality source
- `LOW` — supported by 1 market with limited evidence

**Guidance on ranking**: If two insights have the same evidence breadth,
prefer the one with higher decision impact. An insight that leads to a
specific immediate action outranks an interesting observation with no
clear response.

## Intelligence Gaps

What the analysis cannot see — missing intelligence that would materially
change findings or enable better decisions. Every gap here is an argument for
running additional capabilities.

| Gap | Why It Matters | Decisions It Would Inform | How to Fill It |
|---|---|---|---|
| [missing intelligence — e.g., "No persona cards for UK market"] | [what analysis is limited by this absence] | [which strategic questions it would answer] | [which capability to run — pe-persona-definition, pe-competitor-benchmarking, etc.] |

**Common intelligence gaps to check:**

- Markets with no persona cards (cannot run persona pattern analysis)
- Markets with no competitor profiles (cannot assess competitive position)
- Products with no health checks (cannot assess portfolio health)
- Initiatives with no decision records (decision patterns are incomplete)
- Demand signal reports older than 6 months (signal data may be stale)
- Any market with artifacts in only 1 type (thin cross-reference capability)

## Synthesis Quality Check

Before finalizing the report, verify:

- [ ] Every insight in the Top 5 table cites specific evidence (artifact
      names or markets — not vague references)
- [ ] Confidence levels reflect actual evidence breadth (HIGH = 3+ sources)
- [ ] Systemic issues are distinguished from product-specific issues
- [ ] Intelligence gaps list at least one actionable capability to run
- [ ] No patterns are fabricated — if fewer than 2 data points exist for
      a claimed pattern, downgrade to "observation" or remove
- [ ] Portfolio implications lead to specific next actions or initiative
      recommendations, not generic statements

> **The synthesis is the deliverable.** Analyses 1–5 are the evidence base.
> The synthesis is what the user will act on. Make it specific, prioritized,
> and actionable.
