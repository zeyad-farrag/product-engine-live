## Analysis 1 — Persona Convergence Matrix

### Output Format

```
### Persona Convergence Matrix

| Archetype Name | Core Job-to-be-Done | Pain Point Cluster | Markets Observed | Convergence | Market-Agnostic? |
|---|---|---|---|---|---|
| [Name] | [What they're trying to accomplish] | [Shared frustrations] | [Market A, B, C] | STRONG (5) | YES — product opportunity |
| [Name] | [JTBD] | [Pain cluster] | [Market A, B] | MODERATE (3) | YES — product opportunity |
| [Name] | [JTBD] | [Pain cluster] | [Market A] | WEAK (1) | NO — market-unique |
```

**Convergence Level** = number in parentheses is the count of distinct markets/studies where this archetype appears.
- STRONG: 5+ independent sources
- MODERATE: 3–4 independent sources
- WEAK: 2 independent sources
- SIGNAL (not pattern): 1 source — do not label as pattern

```

### Market-Unique Personas

| Persona | Market | Why Unique | Strategic Note |
|---|---|---|---|
```

## Analysis 6 — Hypothesis Testing

### Hypothesis Testing Report Format

For each hypothesis (user-provided or data-suggested):

```
---

### Hypothesis [N]: "[State the hypothesis as a clear testable claim]"

**Source**: [User-provided / Data-suggested from Analysis 1/2/3/4/5]

#### Supporting Evidence

| Evidence | Source File | Strength |
|---|---|---|
| [Specific finding that supports] | [filename] | STRONG / MODERATE / WEAK |
| [Another supporting finding] | [filename] | MODERATE |

#### Contradicting Evidence

| Evidence | Source File | Strength |
|---|---|---|
| [Specific finding that contradicts] | [filename] | STRONG / MODERATE / WEAK |
| [Another contradiction] | [filename] | WEAK |

#### Evidence Assessment

Supporting: [N strong, N moderate, N weak]
Contradicting: [N strong, N moderate, N weak]
Evidence quality: [Good / Limited / Sparse]

#### Verdict: [SUPPORTED / PARTIALLY SUPPORTED / NOT SUPPORTED / INSUFFICIENT DATA]

**Reasoning**: [2–3 sentences explaining the verdict]
**Confidence**: [HIGH / MEDIUM / LOW]
**Recommended action**: [What to do given this verdict]

---
```

### Verdict Definitions

- **SUPPORTED**: Majority of evidence supports; no strong contradictions
- **PARTIALLY SUPPORTED**: Mixed evidence; hypothesis holds in some contexts but not others
- **NOT SUPPORTED**: Strong contradicting evidence outweighs support
- **INSUFFICIENT DATA**: Less than 2 independent sources either way — cannot test yet

### Data-Suggested Hypothesis Generation

When no user hypothesis is provided, generate 3–5 hypotheses from patterns found in Analyses 1–5:

Look for:
- Persona archetypes appearing in many markets → hypothesis about universal need
- Multi-market competitors with same weakness → hypothesis about market opportunity
- Recurring decision approval factors → hypothesis about what drives success
- Systemic gaps → hypothesis about organizational root cause
- Hidden demand signals → hypothesis about unmet market need

Format: "We hypothesize that [X] because [pattern observed in analyses]."
Then test each using the template above.
